Setting Up venv on Windows

✅ Step 1: Install Python (if not installed)

Download from: https://www.python.org

Important: During installation, check the box “Add Python to PATH”

✅ Step 2: Create a Virtual Environment

# Open Command Prompt or PowerShell.

# Navigate to your folder:

cd project\Test

Then run:

python -m venv test

📁 This will create a folder structure like:


project/
└── Test/
    └── test/
        ├── Scripts/
        └── Lib/

✅ Step 3: Activate the Virtual Environment

1. Using Command Prompt (Recommended in VS Code):

test\Scripts\activate

If it doesn’t work, try:

test\Scripts\activate.bat

✅ It's better to use Command Prompt (cmd.exe) because it doesn’t require changing execution policy like PowerShell.

2. Using PowerShell:

.\test\Scripts\Activate.ps1

❗ If you see this error:

File Activate.ps1 cannot be loaded because running scripts is disabled...

Run this once in PowerShell as Administrator:

Set-ExecutionPolicy -Scope CurrentUser -ExecutionPolicy RemoteSigned

Then activate again:

.\test\Scripts\Activate.ps1

✅ You’ll now see:

(test) C:\project\Test>

This means the virtual environment is active.

✅ Step 4: Install a Module (e.g., requests)

Use this after activating the environment:

pip install requests

✅ Step 5: Deactivate the Environment

To exit the virtual environment:

deactivate
